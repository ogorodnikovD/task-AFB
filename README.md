### Установка и запуск

1.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/ogorodnikovD/task-AFB.git
2.  Перейдите в директорию проекта:
    ```bash
    cd task-AFB
3.  Убедитесь, что в корне проекта присутствует файл clients.csv
4.  Запустите Go-сервер:
    ```bash
    go run ./cmd/web
5.  Отправка запроса к серверу:
    В отдельном терминале выполните Python-скрипт для отправки тестового запроса:
    ```bash
    python request.py
    (Убедитесь, что сервер уже запущен перед выполнением скрипта)
